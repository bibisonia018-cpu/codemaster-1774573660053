# 🔒 تطبيق الدردشة السرية (Secret Chat App - E2EE)

تطبيق دردشة مبني على React.js و Firebase يوفر بيئة تواصل آمنة تعتمد على التشفير من طرف إلى طرف (End-to-End Encryption).

## 🚀 المميزات
- **بدون حسابات**: تسجيل دخول مجهول تماماً (Anonymous).
- **تشفير E2EE**: يتم تشفير رسائلك محلياً في المتصفح باستخدام `AES`، ولا تصل قاعدة البيانات إلا كنصوص مشفرة.
- **تزامن لحظي**: يعتمد على `onSnapshot` في Firebase Firestore.
- **واجهة عربية**: تصميم عصري ومخصص ليكون من اليمين لليسار.
